Исправить алгоритм pagination, чтобы страницы заполнялись максимально и не возникали большие пустые области.

Сейчас система слишком рано переносит абзацы на следующую страницу, что приводит к недозаполненным страницам.

Нужно изменить логику переноса абзацев.

1. Разрешить разрыв абзацев

Абзацы не должны переноситься целиком на следующую страницу.

Удалить правило:

keepParagraphTogether

Вместо этого разрешить разрыв абзаца между страницами.

2. Добавить widow / orphan control

Разрыв абзаца допускается только если соблюдены условия:

минимум 2 строки в конце страницы
минимум 2 строки в начале следующей страницы

Пример:

ПРАВИЛЬНО

Страница 1
строка
строка

Страница 2
строка
строка

НЕПРАВИЛЬНО

Страница 1
строка

Страница 2
остальной абзац
3. Изменить алгоритм pagination

Алгоритм должен работать построчно, а не по абзацам.

Правильная логика:

for each line in paragraph:

    if line fits page:
        place line

    else:
        if linesOnCurrentPage >= 2:
            start new page
            place line
        else:
            move previous line to next page
4. Максимальное заполнение страницы

Страница должна быть заполнена максимально.

Правило:

pageFillTarget = 95%

Если пустая область больше 5–10%, движок должен:

попробовать добавить следующую строку

или перераспределить строки.

5. Разрешить перенос последнего абзаца

Последний абзац на странице должен иметь право начинаться, даже если он не помещается целиком.

Пример:

ПРАВИЛЬНО

строка
строка
строка
начало абзаца
6. Проверка перед переносом

Перед созданием новой страницы движок обязан проверить:

можно ли поместить ещё 1 строку

Если да — строка должна быть размещена.

Почему это исправит твою проблему

После этого:

❌ пустые области исчезнут
❌ страницы будут заполняться
❌ абзацы будут переноситься корректно
❌ widow/orphan сохранится.

Важная вещь, которую ещё нужно добавить

Я вижу по скринам ещё одну проблему.

У тебя очень маленький шрифт:

9.5 pt

Для A5 книги нормальные параметры:

шрифт: 10.5 – 11 pt
line height: 1.35 – 1.45

Сейчас текст слишком плотный.
