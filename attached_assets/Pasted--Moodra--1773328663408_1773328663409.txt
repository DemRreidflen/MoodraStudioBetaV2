Разработай интегрированный модуль верстки книг внутри платформы Moodra, который превращает процесс подготовки рукописи в полноценный визуальный конструктор книги, аналогичный Figma, но специально оптимизированный для книжной типографики.

Этот модуль должен быть частью существующей системы Moodra и работать с уже созданными книгами пользователя.

Главная цель:
позволить автору визуально спроектировать и сверстать книгу, чтобы итоговый PDF выглядел 1:1 так же, как макет в редакторе.

1. Основная концепция интерфейса

Создай Moodra Book Layout Studio — визуальное пространство, где пользователь видит книгу как физический объект.

Интерфейс должен напоминать комбинацию:

• Figma (гибкость дизайна)
• Word (работа с текстом)
• InDesign (книжная типографика)

Пользователь должен работать не с текстовым документом, а с настоящими страницами книги.

2. Визуальное представление книги

Центр интерфейса — Book Canvas.

Он отображает книгу как реальные страницы.

Функции:

• просмотр книги сверху как макета
• двухстраничный режим (spread view)
• режим одной страницы
• превью всей книги
• прокрутка страниц
• листание как настоящей книги

Пользователь может:

• листать страницы
• приближать и отдалять масштаб
• переходить к конкретной странице
• видеть нумерацию страниц

7. Автоматическое оглавление

Каждая книга обязательно содержит страницу Contents.

Система автоматически создаёт оглавление.

Оно должно:

• обновляться автоматически

9. Управление страницами

Система должна позволять:

• добавлять страницы
• удалять страницы
• менять порядок
• дублировать страницы
• применять шаблоны страниц

Также должна быть панель структуры книги.

Она показывает:

главы

подглавы

страницы

10. Превью печати

Перед экспортом пользователь может открыть Print Preview Mode.

Этот режим показывает:

• реальные отступы
• bleed
• safe margins
• как книга будет выглядеть после печати

11. Экспорт

Moodra должна экспортировать книгу в:

PDF (основной формат)

Файл должен быть:

• типографически корректный
• готовый для печати
• 1:1 совпадающий с макетом

Также возможен экспорт:

EPUB

MOBI

DOCX

12. Гарантия WYSIWYG

Очень важно:

Редактор должен работать по принципу WYSIWYG.

Что пользователь видит на экране —
точно так же будет выглядеть PDF.

Без изменений шрифтов
Без сдвига блоков
Без изменения расстояний.

13. Гибкость уровня дизайна

Moodra Layout Studio должна позволять:

• менять сетку страниц
• настраивать колонки
• менять расстояния между строками
• использовать декоративные элементы
• создавать уникальные стили страниц

Это должно быть полноценное пространство для дизайна книги, а не просто редактор текста.

14. Производительность

Даже если книга содержит:

300–500 страниц

редактор должен:

• работать быстро
• плавно прокручивать страницы
• мгновенно обновлять стили

15. Интеграция с Moodra AI

Layout Studio должна быть связана с AI.

AI может:

• предложить стиль верстки
• автоматически оформить книгу
• оптимизировать типографику
• улучшить структуру главы
