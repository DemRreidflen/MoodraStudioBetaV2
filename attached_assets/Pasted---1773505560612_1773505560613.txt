Отлично, текущий движок верстки уже работает хорошо, поэтому важно не менять и не ломать существующую логику layout engine. Нужно добавить только несколько улучшений поверх уже работающей системы.

1. Корректный перенос слов

Когда слово переносится на новую строку или страницу, необходимо автоматически добавлять дефис (черточку) в конце первой части слова.

Требования:

дефис должен появляться только при реальном переносе слова

перенос должен происходить по правилам языка (словарные точки переноса)

нельзя разрывать слово случайным образом

нельзя допускать перенос без дефиса

нельзя допускать двойной дефис или лишние символы

Пример:

перенос
перено-
сится
2. Два режима предпросмотра книги

Добавить удобное переключение между двумя режимами просмотра:

Режим 1 — Одна страница

отображается одна страница

страница центрирована

выглядит как отдельная книжная страница

Режим 2 — Разворот

отображаются две страницы рядом

выглядит как открытая книга

сохраняется расстояние между страницами (gutter)

Важно:

переключение режима не должно менять верстку

это только визуальный режим просмотра

page breaks, строки и переносы должны оставаться такими же.

3. Корректное скачивание PDF

Сейчас файл открывается как онлайн-страница. Нужно изменить поведение.

Требования:

при нажатии Download PDF файл должен скачиваться

PDF должен выглядеть точно так же, как в предпросмотре

сохранить:

размеры страниц

поля

переносы слов

разрывы страниц

нумерацию страниц

оформление глав

типографику

То есть система должна работать по принципу:

Preview = Exported PDF

PDF должен генерироваться из того же page layout, который используется для предпросмотра.

Важное условие

Эти изменения должны быть добавлены поверх текущего движка, без переписывания существующей системы верстки.

Не изменять:

pagination logic

page scoring

page templates

footer safe zone

paragraph layout

Нужно только:

добавить корректное отображение дефиса при переносе слова

реализовать переключение режима предпросмотра (страница / разворот)

сделать корректное скачивание PDF.
